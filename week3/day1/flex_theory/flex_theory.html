Understanding Flow Layout

What is Flow Layout?

When it comes to layout, CSS acts as a collection of mini-languages, each designed for different layout tasks. These are known as layout modes, and there are seven distinct ones. In this lesson, we'll focus on Flow Layout, the default layout mode.

A plain HTML document with no CSS applied exclusively uses Flow Layout. You can think of it as the "Microsoft Word" layout algorithm, intended for document-like content organization.

Types of Elements in Flow Layout

Flow Layout categorizes elements into two main types:

  • Block elements: These include <div>, <header>, <p>, and similar tags. They stack vertically, taking up the full width of their container by default.
  • Inline elements: These include <span>, <a>, and <strong>. They flow horizontally, wrapping to the next line when needed.

Changing the Display Property

Each HTML element has a default display type, but we can modify it using the display property. For example:

a.nav-link { display: block; }

The above code transforms a link from its default inline display to block.

Behavior of Inline and Block Elements

Inline Elements

Inline elements are "go-with-the-flow" by nature. They:

  • Can only adjust spacing horizontally using margin-left and margin-right.
  • Do not support layout properties like width or height.

However, some elements, like <img>, are exceptions. These are called replaced elements. They behave like inline elements but can accept dimensions and impact layout:

img { width: 300px; height: 200px; }

Block Elements

Block elements are "greedy" when it comes to space. They:

  • Always expand to fill the full width of their container, even if their content only requires less space.
  • Stack vertically, creating a new "block" for each element.

Hello World

This heading takes up the full width of its container.

Inline-Block: A Hybrid

The inline-block value blends the behaviors of inline and block elements. It allows you to:

  • Place elements inline, like text.
  • Apply layout properties like width and margin-top.
strong { display: inline-block; width: 100px; margin-top: 20px; text-align: center; }

Key Takeaways

  • Flow Layout is the default and works well for document-style layouts.
  • Block elements stack vertically and take up the full width of their container.
  • Inline elements flow horizontally and do not support width or height.
  • inline-block allows inline placement with block-level layout capabilities.
